\doxysubsection{Detailed Description}
general exception of the \doxyref{basic\+\_\+json}{p.}{classnlohmann_1_1basic__json} class 

This class is an extension of {\ttfamily std\+::exception} objects with a member {\itshape id} for exception ids. It is used as the base class for all exceptions thrown by the \doxyref{basic\+\_\+json}{p.}{classnlohmann_1_1basic__json} class. This class can hence be used as \char`\"{}wildcard\char`\"{} to catch exceptions.
